# Linux From Scratch Archipelago

A Linux From Scratch Manual Archipelago. Best played with a multi-world
multi-game Archipelago.

This world has not been tested and traps are not implemented. Please report all
cyclic requirements chain for building software and dependencies locked behind
programs that indirectly requires them.
