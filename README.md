# Linux From Scratch Archipelago

A Linux From Scratch Manual Archipelago. Best played with a multi-game
multi-world Archipelago setup.

This world has not been tested and traps are not implemented. Please report all
cyclic dependencies for programs or libraries locked behind other programs or
libraries that indirectly requires them.
